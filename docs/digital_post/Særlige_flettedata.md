# Særlige Flettedata

## Intro

Nogle kolonner i den uploadede flettedata har særlige funktioner i OpenPostbud.

Du kan se en oversigt over dem her, og læse mere i detaljer længere nede.

| Navn          | Beskrivelse               | Obligatorisk |
| :------------ | ------------------------- | ------------ |
| Memo Modtager | Cpr-nummer på modtager    | Ja           |
| Memo Label    | Overskrift/emne på brevet | Ja           |

### Memo Modtager

Denne kolonne indeholder CPR-numre på modtagere af brevene.

### Memo Label

Denne kolonne indeholder overskrifterne på brevene. Dette bliver vist
som overskrift i modtagerens Digitale Postkasse.
Selvom man måske vil have samme overskrift på alle breve i forsendelsen,
skal man stadig have denne for hvert brev.
